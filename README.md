## Project Description

**Live Demo:** https://arifbuilder.github.io/project-8--nodedesk/

This project was created as part of my web development learning journey. The goal was to recreate a given website as accurately as possible to practice my frontend development skills.

The entire project was built manually using **HTML** and **CSS**, including a fully responsive layout. No AI tools or vibe coding were used—every part of the code was written by me to better understand web design, layouts, and responsive development.

While the project closely matches the original design, there are still a few minor issues that I plan to improve as I continue learning.
